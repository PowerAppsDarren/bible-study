# Your chapter studies

Mirror the public repo's layout so navigation feels the same:

```
scripture/
└── NN-BookName/            ← e.g. 01-Genesis, 23-Isaiah, 66-Revelation
    ├── BookName-NN/        ← e.g. Genesis-01, Isaiah-11
    │   └── notes.md        ← your reflections on that chapter
    └── sources/            ← raw inputs that fed a study (transcripts, exports)
```

Your `notes.md` is *yours* — reflection, application, teacher-voice, the
speculative connections that wouldn't pass the public repo's bar. Anything
factual you want to contribute back goes to the public chapter README via a
Pull Request.

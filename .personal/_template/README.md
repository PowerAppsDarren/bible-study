# Your private study space

This is **your** folder. It's a separate, private git repo — not part of the
public Bible-study repo, and not on GitHub. Push it wherever you like (your own
private Forgejo, a private GitHub repo, a NAS, or keep it local only).

Organize it however helps you. The layout below mirrors the public repo so
things are easy to find, but it's only a suggestion — your space, your call.

```
<this folder>/
├── profile.md            ← who you are, what you're studying
├── reading-plan.md       ← your plan / where you are
├── journal/
│   └── YYYY-MM-DD.md      ← daily entries
├── prayer/
│   └── prayer-list.md
└── scripture/            ← chapter studies, mirroring the public scripture/ layout
    └── 01-Genesis/
        └── Genesis-01/
            └── notes.md
```

## Two halves of every study

- **Your half** (reflection, application, what stirred *you*, prayer) lives
  here, in this private repo. Everything is welcome here.
- **The factual half** (what the text says, means, and connects to) can be
  offered back to the *public* repo as a Pull Request — see the main repo's
  `CONTRIBUTING.md` for the bar it has to clear.

## Daily rhythm

read a chapter → write here → `git commit` → push to your own git.

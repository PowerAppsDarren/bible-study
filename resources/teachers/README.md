# Teacher Resource Libraries

Curated, copyright-safe reference material for the Bible teachers modeled as agents in this repo. Each teacher has a folder here whose name matches their agent slug in `.claude/agents/` (minus the `teacher-` prefix).

## Convention

```
resources/teachers/<kebab-name>/
  README.md     # overview, agent link, index of this folder
  links.md      # external resources — URLs, free/paid flag, access date
  notes.md      # structured, transformative study notes on the teacher's frameworks
  quotes.md     # short verbatim quotes WITH attribution + source
```

**Shared-layer copyright rule (from `CONTRIBUTING.md`):** this folder is for *pointers, transformative summaries, and short attributed quotes only*. No full-text article copies, transcripts, downloaded PDFs of copyrighted talks, or scanned book pages. Full-text personal copies belong in `.personal/<your-email>/` and must not be redistributed.

Every external item carries a source URL, an access date, and a free/paid flag.
